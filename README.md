## 概要

Redux Toolkit のサンプルです。  
バックエンドは Spring Boot で REST API を作成しています。

以下の要素を含みます：

**フロント (React)**
* Basics
  - TypeScript
  - Custom Hook
  - 再レンダリングの最適化 (memo, useCallback)
* State management
  - Redux Toolkit
* Styling
  - CSS Modules
  - Material UI
  - React Icons
* Routing
  - React Router
* API
  - axios

**バック (Spring Boot)**
* REST API
* Swagger UI (Open API ドキュメント)
